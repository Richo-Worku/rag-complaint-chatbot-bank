# RAG Complaint Chatbot

AI-powered complaint intelligence system using Retrieval-Augmented Generation (RAG) on CFPB financial complaint data.

## Features

- Semantic search over 1.3M complaint chunks
- Product-level filtering (Credit Cards, Loans, etc.)
- RAG-based explanation of complaint trends
- Analytics for compliance and product teams

## Tech Stack

- Python
- ChromaDB / FAISS
- Sentence Transformers
- FastAPI
- Streamlit
